# SisOp

### Requerimentos para compilar: 
- gcc instaldo

## Como rodar o código: 

##### Em c:
> Primeiro execute o comando a seguir, sendo **2.cpp** o nome do arquivo e **./usefork** o diretório de onde estará o arquivo compilado;
~~~javascript
g++ ./2.cpp -o ./usefork
~~~
> Em seguida, execute o seguinte comando:
~~~javascript
./usefork
~~~

##### Em python:
~~~python
python run 3.py
~~~
